# Monte Carlo Integration

## Autorzy
Karolina Gralak, Wiktor Chmiel, Marek Makowski

## Opis projektu
Projekt dotyczy implementacji metod Monte Carlo do wyznaczania całek oznaczonych na przedziałach prostokątnych. W ramach pracy porównano dwa podejścia do generowania punktów wykorzystywanych w estymacji całek:
- generator wbudowany `Rnd()` w VBA,
- sekwencje Haltona.

## Cel projektu
Celem projektu było zbadanie skuteczności metod Monte Carlo w przybliżaniu wartości całek podwójnych oraz porównanie dokładności klasycznego losowania punktów z podejściem opartym na deterministycznych sekwencjach Haltona.

## Zakres projektu
- implementacja algorytmu Monte Carlo z użyciem funkcji `Rnd()`,
- implementacja funkcji generującej liczby Haltona,
- wyznaczanie całek podwójnych dla wybranych funkcji,
- porównanie błędów obu metod,
- analiza wpływu liczby punktów `N` na dokładność estymacji.

## Wykorzystane narzędzia
- **Excel**
- **VBA**
- metody numeryczne
- metoda Monte Carlo
- sekwencje Haltona
